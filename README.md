# 计算机科学课程笔记（Computer-Science-Notes）

大学计算机科学课程学习笔记与资料仓库。

本仓库用于集中整理计算机本科阶段常见课程内容，包括：
- 课程笔记
- 实验代码与实验报告
- 课件要点整理
- 习题与题解
- 高质量外部学习资源

仓库开源并欢迎 Pull Request 共建。

## 项目目标

- 建立长期可维护的计算机知识库。
- 逐步覆盖本科阶段核心计算机课程。
- 提供统一模板，降低贡献门槛。
- 保持内容结构化、可检索、易协作。

## 参与贡献

1. 阅读 [CONTRIBUTING.md](CONTRIBUTING.md)。
2. 从 `main` 分支创建新分支。
3. 按目录规范新增或改进内容。
4. 使用 PR 模板提交 Pull Request。

课程规划可查看：
- [文档/课程索引.md](文档/课程索引.md)
- [文档/路线图.md](文档/路线图.md)

## 推荐贡献类型

- 为某门课程补充完整章节笔记。
- 补充可复现的实验步骤与代码。
- 增加带解析的高质量习题与题解。
- 修订现有笔记中的不清晰或错误内容。
- 增补书籍、论文、视频、官方文档等资源索引。
