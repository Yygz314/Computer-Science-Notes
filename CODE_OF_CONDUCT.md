# 行为准则

## 我们的期望

为了保持良好的协作氛围，贡献者应当：
- 保持尊重与专业。
- 讨论观点而非攻击个人。
- 接受并欢迎建设性反馈。
- 技术结论尽量给出依据。
- 共同维护包容、友好的学习环境。

以下行为不可接受：
- 骚扰、歧视或仇恨言论。
- 人身攻击、威胁、羞辱。
- 泄露他人隐私信息。
- 持续的恶意争论或破坏性行为。
- 垃圾信息或恶意内容。

## 适用范围

本行为准则适用于：
- Issues
- Pull Requests
- 讨论区与 Review 评论
- 其他与本项目相关的沟通场景

## 执行措施

维护者可根据情况采取措施，包括：
- 编辑或删除不当评论。
- 关闭破坏性 Issue/PR。
- 临时或永久限制参与权限。

## 举报方式

如发现违规行为，请私下联系维护者并提供：
- 事件简述
- 相关链接或截图
- 有助于判断的上下文信息

我们会尽快并尽可能公平地处理举报。
