## 1. 系统简介

**学生成绩管理系统**是一款面向教师的桌面端应用，提供以下核心能力：

- 学生信息的增删改查（支持学号、姓名、性别、班级、分数五个字段）
- 多条件组合高级搜索，支持模糊匹配
- Excel 批量导入 / 导出（`.xlsx` 格式）
- 成绩统计分析（等级分布、各班均分排名）
- 用户账号管理（密码修改、多账号切换）
- 内置计算器与文本编辑器小工具

## 2. 运行方式

**前提条件**：已安装 Java 运行环境（JRE 8 或以上版本）。

双击运行以下文件：

```
studentmanager\build\libs\studentmanager-1.0-SNAPSHOT.jar
```

或在终端执行：

```bash
java -jar studentmanager-1.0-SNAPSHOT.jar
```

> 首次运行时，程序会在 JAR 所在目录自动创建数据库文件 `students.db`，并初始化默认管理员账号。

## 5. 学生管理

### 5.1 添加学生

**入口**：工具栏 `+ 添加` 按钮 / 菜单栏 学生管理 → 添加学生

在弹出的对话框中填写以下信息：

| 字段 | 是否必填 | 说明 |
|------|----------|------|
| 学号 | **必填** | 正整数，系统检查唯一性，重复则提示重新输入 |
| 姓名 | **必填** | 任意字符串 |
| 性别 | 选填 | 下拉选择：男 / 女 / 不填 |
| 班级 | 选填 | 任意字符串，如 "计算机2301" |
| 分数 | **必填** | 0 ~ 150 之间的整数 |

填写完毕后点击 **确认添加** 保存，点击 **取 消** 放弃。

> **注意**：学号一旦创建不可修改，请仔细核对后再提交。

---

### 5.2 编辑学生

**入口**：
- 工具栏 `✎ 编辑` 按钮
- 双击表格中的目标行
- 右键菜单 → ✎ 编辑学生
- 菜单栏 学生管理 → 编辑学生

> 使用工具栏按钮或菜单入口时，须先在表格中选中目标行。

在编辑对话框中：

- **学号**：灰底灰字，**只读，不可修改**，用于标识学生身份
- **姓名、性别、班级、分数**：均可修改

修改后点击 **保存修改**，若无任何字段发生变化则不执行数据库写操作。

---

### 5.3 删除学生

**入口**：工具栏 `✕ 删除` 按钮 / 右键菜单 / 菜单栏

删除前系统会弹出确认对话框：

```
确认删除学生【张三（学号 10001）】？
```

点击 **是** 执行删除，操作**不可撤销**；点击 **否** 取消。

## 6. 数据搜索与筛选

### 6.1 快速搜索

工具栏搜索区包含：

- **下拉框**：选择搜索字段（全部 / 按学号 / 按姓名 / 按班级）
- **输入框**：输入关键词
- **搜索** 按钮（或按 `Enter` 键）：执行搜索
- **全部** 按钮：清空搜索，显示所有数据

> 选择"全部"时，无论输入框内容如何，均显示所有记录。

---

### 6.2 高级搜索

**入口**：菜单栏 学生查询 → 高级搜索

支持以下条件的任意组合：

| 条件 | 说明 |
|------|------|
| 学号 | 包含该数字的学号 |
| 姓名 | 精确或模糊匹配 |
| 性别 | 下拉选择（不限 / 男 / 女） |
| 班级 | 精确或模糊匹配 |
| 分数区间 | 最低分 ~ 最高分（可只填一端） |
| 模糊匹配 | 勾选后姓名和班级使用 `LIKE` 模糊查询，否则精确匹配 |

- **执行搜索**：将结果回显至主界面表格
- **重置条件**：清空所有输入，恢复默认值
- **关 闭**：关闭对话框，主界面数据保持当前状态

## 7. 数据导入与导出

### 7.1 导出 Excel

**入口**：工具栏 `↓ 导出` / 菜单栏 学生管理 → 导出 Excel

1. 系统弹出"另存为"文件选择器，默认文件名包含时间戳，格式为 `.xlsx`
2. 选择保存路径，点击保存
3. 导出成功后弹出提示，显示文件完整路径

**导出内容**：当前表格中显示的所有记录（若处于搜索结果状态，则只导出搜索结果）。

**Excel 格式**：

| 列 | 内容 |
|----|------|
| A | 学号 |
| B | 姓名 |
| C | 性别 |
| D | 班级 |
| E | 分数 |

第 1 行为加粗蓝色表头，数据从第 2 行开始，所有单元格居中并带细边框。

---

### 7.2 导入 Excel

**入口**：工具栏 `↑ 导入` / 菜单栏 学生管理 → 导入 Excel

**操作步骤**：

**第一步**：选择文件  
弹出文件选择器，选择 `.xlsx` 格式的 Excel 文件。

**第二步**：选择重复处理策略  

| 选项 | 行为 |
|------|------|
| 覆盖原有数据 | 若学号已存在，用文件中的数据覆盖数据库中的记录 |
| 跳过该行 | 若学号已存在，忽略该行，不做任何修改 |
| 取消导入 | 放弃本次导入操作 |

**第三步**：系统自动处理并显示结果汇总：

```
导入完成！
新增 10 条   更新 3 条   跳过 2 条

以下行存在问题：
• 第 5 行：分数超出范围（200）
• 第 8 行：姓名为空
```

**导入文件格式要求**：

- 文件格式：`.xlsx`（Excel 2007 及以上）
- 第 1 行为表头（内容不做限制，会自动跳过）
- 从第 2 行起为数据行，列顺序须与导出格式一致：**学号、姓名、性别、班级、分数**
- 学号须为正整数，分数须在 0 ~ 150 之间

> **提示**：可以先导出现有数据作为模板，再在模板基础上填写新数据后导入。

## 8. 成绩统计分析

**入口**：菜单栏 学生查询 → 成绩统计

统计对话框包含三个面板：

### 总体概况

以四个彩色数值卡展示当前数据集的整体情况：

```
┌──────────┐ ┌──────────┐ ┌──────────┐ ┌──────────┐
│    86    │ │   78.5   │ │    99    │ │    45    │
│  总人数  │ │  平均分  │ │  最高分  │ │  最低分  │
└──────────┘ └──────────┘ └──────────┘ └──────────┘
```

### 等级分布

| 等级 | 分数段 | 人数 | 占比 |
|------|--------|------|------|
| 优秀 | 90 ~ 100 | N | xx% |
| 良好 | 75 ~ 89 | N | xx% |
| 合格 | 60 ~ 74 | N | xx% |
| 不合格 | 0 ~ 59 | N | xx% |

等级列与主表格分数着色方案一致。

### 各班成绩

按班级分组，显示各班的人数、均分、最高分、最低分，并按均分**从高到低排序**。

> **注意**：统计范围是主界面当前显示的数据，若主界面处于搜索状态，则只统计搜索结果，而非全部数据。

## 10. 小工具

### 10.1 计算器

**入口**：菜单栏 小工具 → 计算器

支持四则运算和括号，可通过鼠标点击或键盘输入：

| 功能 | 说明 |
|------|------|
| `C` | 清空所有内容 |
| `←` | 删除最后一个字符 |
| `( )` | 智能括号（自动判断插入左括号还是右括号） |
| `±` | 切换正负号 |
| `=` / `Enter` | 计算结果 |
| `Esc` | 等同 C，清空 |

**键盘快捷键**：数字键、`+` `-` `*` `/` `%` `.` `(` `)` `Enter` `Backspace` `Esc` 均可直接使用。

**示例**：

```
输入：(3+5)*2-1
结果：15
```

---

### 10.2 文本编辑器

**入口**：菜单栏 小工具 → 文本编辑器

支持 UTF-8 编码的文本文件读写，完整支持中文输入。

#### 主要功能

| 功能 | 快捷键 |
|------|--------|
| 新建 | `Ctrl + N` |
| 打开 | `Ctrl + O` |
| 保存 | `Ctrl + S` |
| 另存为 | 菜单 格式 → 另存为 |
| 撤销 | `Ctrl + Z` |
| 重做 | `Ctrl + Y` |
| 剪切 | `Ctrl + X` |
| 复制 | `Ctrl + C` |
| 粘贴 | `Ctrl + V` |
| 全选 | `Ctrl + A` |
| 查找 / 替换 | `Ctrl + H` |

#### 查找 / 替换

- **查找下一个**：高亮所有匹配项，并跳转到下一处；到达末尾后从头继续
- **替换**：替换当前选中的匹配项，并跳到下一处
- **全部替换**：一次性替换所有匹配项，并显示替换数量
- **区分大小写**：勾选后严格匹配大小写

#### 其他功能

- **左侧行号面板**：实时显示行号
- **底部状态栏**：实时显示字符数、字数、行数、修改状态
- **光标位置**：右下角显示当前行列号
- **字体设置**：格式 → 字体设置，可选择任意系统字体和字号
- **自动换行**：格式 → 自动换行，切换换行模式
- **关闭保护**：若有未保存内容，关闭时弹出"是否保存"确认

## 11. 常见问题

**Q：双击 JAR 文件没有反应，怎么办？**  
A：确认已安装 Java（版本 ≥ 8），或在命令行执行 `java -jar studentmanager-1.0-SNAPSHOT.jar`。

**Q：数据存储在哪里？**  
A：程序运行后，数据库文件 `students.db` 自动生成于 JAR 文件所在目录，删除该文件会清空所有数据，请注意备份。

**Q：导入 Excel 时提示"读取文件失败"？**  
A：确认文件格式为 `.xlsx`（不支持旧版 `.xls` 格式）。可将 `.xls` 文件在 Excel 中另存为 `.xlsx` 后再导入。

**Q：忘记密码怎么办？**  
A：目前无密码找回功能。可直接用 SQLite 数据库工具（如 DB Browser for SQLite）打开 `students.db`，在 `users` 表中手动修改 `password` 字段。

**Q：添加学生时提示"学号已存在"，但表格中看不到该学生？**  
A：可能该学生处于当前搜索结果之外。点击工具栏 **全部** 按钮刷新显示所有记录后再查找。

**Q：文本编辑器输入中文乱码？**  
A：编辑器已采用支持中文的字体（微软雅黑），可在 格式 → 字体设置 中确认或更换字体。若保存后用其他编辑器打开乱码，请确保对方编辑器也使用 UTF-8 编码打开。
